MultiCam Simulator

Prototype · In Development

A training simulator for multicamera live production, built in Unity. The idea came from my experience as a TV producer and university instructor in media production — I kept seeing the same problem: students struggle to learn multicamera workflows without access to real studio equipment.

This prototype simulates a 10-camera concert setup with a video switcher interface. It's a tool for understanding the basics before stepping into a real control room.

Download for macOS

Requires macOS 15.0 or later (Apple Silicon + Intel)

What It Does

Multiview wall with 10 camera feeds, Preview and Program monitors — like a real broadcast control room

Simplified video switcher with Preview/Program rows that swap roles on cut — matching real broadcast logic

10-camera concert setup including crane and dolly shots

Controls

Keyboard

1 2 ... 0 Switch camera to Program (0 = camera 10)
LeftShift + 1-0 Send camera to Preview
Enter Cut — switch Preview to Program
Space Pause simulation
Esc Return to start

Mouse

Click any camera monitor to send it to Program. Click the T-Bar to cut.

Tech

Unity 3D C# macOS (Apple Silicon)
